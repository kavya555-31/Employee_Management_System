# Employee Management System

## Project Overview

Employee Management System is a full-stack web application developed using Python Flask and SQLite. It helps administrators manage employee records efficiently through a user-friendly interface.

## Features

* Admin Login Authentication
* Add Employee
* View Employees
* Update Employee Salary
* Delete Employee
* Search Employee by ID
* Search Employees by Department
* Salary Analytics Dashboard
* Employee Statistics
* Employee Photo Upload
* SQLite Database Integration

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Bootstrap

## Project Structure

* Frontend: HTML, CSS, Bootstrap
* Backend: Python Flask
* Database: SQLite

## How to Run

1. Install Python.
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open:
   http://127.0.0.1:5000
